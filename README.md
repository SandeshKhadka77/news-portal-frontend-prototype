# News Portal – Frontend Prototype

A responsive and modern Nepal-based news portal frontend prototype built using **HTML, CSS, and JavaScript only**.  
This project is designed as a **static prototype** to demonstrate UI/UX, layout structure, and frontend logic.  
Future versions will upgrade this into a **fully functional news portal**.

##  Project Status
 **Prototype Version**  
- Manual news content written directly in HTML  
- No backend or API integration (yet)  
- Planned upgrade to dynamic and full-featured system  

---

##  Features
- News content (Politics, Sports, Economy, Environment, Tourism)
- Clean and professional UI design
- Long-scroll news layout (real news portal feel)
- Responsive design (mobile, tablet, desktop)
- Modal-based article preview
- Clean, and readable code
- Separate HTML, CSS, and JavaScript files

---

##  Tech Stack
- **HTML5** – Structure
- **CSS3** – Styling & responsiveness
- **Vanilla JavaScript** – UI logic and interactions
